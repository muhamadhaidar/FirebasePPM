A modern, iOS-style habit tracker built with React Native Expo and Firebase.

## Features
- **Cloud Sync**: Real-time data persistence with Firestore.
- **Interactive UI**: Glassmorphic styling and responsive animations (Hover/Click).
- **Gamification**: Streak tracking and daily progress visualization.
- **Cross-Platform**: Optimized for Android, iOS, and Web.

## Tech Stack
- React Native (Expo)
- TypeScript
- Firebase Firestore
- Jest (Testing)

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Setup Firebase**
   - Create `src/config/firebaseConfig.ts` with your credentials.

3. **Run App**
   ```bash
   npm start
   ```

4. **Run Tests**
   ```bash
   npm test
   ```

## Structure
- `src/screens`: Main application screens.
- `src/components`: Reusable UI components.
- `src/contexts`: State management.
